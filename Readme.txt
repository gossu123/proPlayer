Micro_project
Micro_project is a program made using c++. It analysis the sensor data from mobile. It can find roll, pitch and yaw from Accelerometer, Magnetometer data. It can also find the number of steps taken by a person to reach a destination and distance.

To run the project use following commands
           ~$ cd path/to/Micro_project
/Micro_project$ make
/Micro_project$ ./sensor-demo
To see the lcov coverage results
Run Micro_project/results/index.html
To see the documentation
Run Micro_project/docs/html/index.html
